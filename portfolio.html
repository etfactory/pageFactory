The most Difficult Project

studentSystem Project

프로젝트 개요

studentSystem 프로젝트는 개인적으로 진행한 프로젝트로, 에브리타임 어플리케이션을 모티브하여 제작한 어플리케이션 프로젝트입니다.

이 프로젝트는 안드로이드와 iOS에서 모두 사용할 수 있는 크로스 플랫폼 어플리케이션으로, Flutter를 활용하여 개발하였습니다.

발생한 어려움

Flutter를 처음 접하는 상황에서, 다양한 기능을 구현하는 과정에서 어려움을 겪었습니다.

특히, Android와 iOS에서의 UI/UX 차이로 인해 동일한 기능을 구현하는 데 있어 각각의 플랫폼에 맞는 최적화를 고려해야 했습니다.

또한, SQLite 데이터베이스를 활용하여 데이터를 저장하고 불러오는 과정에서, 데이터베이스 구조 설계와 쿼리 작성에 어려움을 겪었습니다.

마지막으로, 프로젝트의 규모가 예상보다 커져 따라 코드 관리와 유지보수에 대한 고민이 많아졌습니다.

해결 방법

Flutter의 공식 문서와 다양한 자료를 통해 Flutter의 기본 개념과 위젯을 학습하였습니다.

또한 안드로이드와 iOS의 네이티브 환경을 이해하면서 각 플랫폼에 맞는 UI/UX를 구현하기 위해 노력하였습니다.

SQLite 데이터베이스를 활용하기 위해, 데이터베이스 구조를 미리 설계하고, 필요한 쿼리를 사전에 작성하여 테스트하는 방법을 사용하였습니다.

프로젝트의 규모가 커짐에 따라, 코드 관리와 유지보수를 위해 Git을 활용하여 버전 관리를 하는 방식으로 해결했습니다.
